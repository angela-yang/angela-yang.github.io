Projects

View Project!

Paint-A-Hike

October 2025

A project that allows the user to draw a sketch of their dream view. The web app then generates a real-world location that is the closest match to their sketch, bringing their dream hike to life. Built using Next.js, TailwindCSS, Python, etc.

My Art Shop!

September 2025 - Ongoing

This project was created to market my artwork. The website is designed like a physical shop to make the shopping experience more exciting and interactive. Created using Next.js, TailwindCSS, and Stripe.

View Project!

Air Quality Health

August 2023

A project that shows users their local air quality index, and percentage of different elements. It can also take in personalized health information to provide personalized feedback on how users can safeguard their wellbeing.

View Project!

Bear Go Brr

February 2024

A project that incentivizes users to reduce their carbon footprint. Through this app, the user adopts a pet polar bear, which they must keep safe from icebergs melting due to their carbon footprint.

View Project!

Conscious Closet

June 2023

This project was developed to motivate users to stay sustainable, using a "K Nearest Neighbors" model to find cheap, accessable, and eco-friendly alternatives to fast fashion. Created using HTML, CSS, Python, and the Flask API.

View Project!

Quizble

November 2024

A bubble-themed study app that allows you to review and study flashcards, offering fun games and quizzes. Created using Express.js, Node.js, Vite, Chakra UI, and hand-drawn illustrations.

View Project!

Crane Game

August 2023

A 2D movement game in which you search for ten paper cranes in the dark. Each crane expands your field of sight. Developed using p5.js, HTML, and CSS. Illustrations done on Procreate, and music found on Youtube.
